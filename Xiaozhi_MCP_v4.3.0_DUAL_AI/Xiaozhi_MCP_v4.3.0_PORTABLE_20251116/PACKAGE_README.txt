🚀 HƯỚNG DẪN NHANH (5 PHÚT)
───────────────────────────────────────────────────────────────────────────

BƯỚC 1: Giải nén package vào thư mục bất kỳ

BƯỚC 2: Nhấp đúp INSTALL.bat → Đợi cài dependencies (1-2 phút)

BƯỚC 3: Lấy Xiaozhi Token
   • Truy cập: https://xiaozhi.me
   • Đăng nhập → Profile → Copy JWT token

BƯỚC 4: Nhấp đúp START.bat → Browser tự mở http://localhost:8000

BƯỚC 5: Dashboard → Click ⚙️ → Dán token → Lưu

🎉 HOÀN TẤT! Giờ bạn có thể ra lệnh cho AI!

🛠️ TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────

❌ Lỗi "Python not found":
   → Cài Python: https://www.python.org/downloads/
   → Check "Add to PATH" khi cài

❌ Server không khởi động:
   → Chạy CHECK.bat để xem lỗi
   → Cài lại: pip install -r requirements.txt --force-reinstall

❌ Không kết nối Xiaozhi:
   → Kiểm tra JWT token
   → Kiểm tra Internet
   → Xem log trong Dashboard

❌ Nhạc không phát:
   → Kiểm tra file trong music_library/
   → Cài Windows Media Player

Chi tiết: Xem README.md phần Troubleshooting
