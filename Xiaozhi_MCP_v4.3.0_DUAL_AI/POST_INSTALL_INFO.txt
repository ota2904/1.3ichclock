🛠️ KHẮC PHỤC SỰ CỐ
══════════════════════════════════════════════════════════════════

❌ "Python not found"
   → Cài Python 3.11+ từ: https://www.python.org/downloads/
   → Nhớ chọn "Add Python to PATH"

❌ "Port 8000 already in use"
   → Đóng ứng dụng đang dùng port 8000
   → Hoặc chạy: taskkill /F /IM python.exe

❌ "Module not found"
   → Chạy: AUTO_INSTALL_DEPENDENCIES.bat
   → Hoặc: pip install -r requirements.txt

❌ Server không khởi động
   → Chạy CHECK.bat để xem lỗi chi tiết
   → Kiểm tra logs trong thư mục logs/
